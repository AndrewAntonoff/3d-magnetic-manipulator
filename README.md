\# 3D Magnetic Levitation Manipulator



Проект манипулятора с магнитной левитацией для CAD-систем и авиасимуляторов.



\## Структура проекта

\- `/hardware` - схемы, платы, 3D-модели

\- `/firmware` - прошивка для шара (Zephyr) и подиума (STM32CubeIDE)

\- `/software` - PC-софт для тестирования и калибровки

\- `/docs` - документация

\- `/schematics` - электрические схемы

\- `/3d-models` - модели для печати
