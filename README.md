# 3D Magnetic Manipulator Project

## Описание
Система магнитной левитации для управления шаром в 3D пространстве с использованием 12 катушек и 5 датчиков магнитного поля.

## Аппаратная часть
- Контроллер: STM32H750VBT6
- Катушки: 12 шт (8 верхний ряд, 4 нижний ряд)
- Датчики: 5× MLX90393 (SPI)
- Питание: 12В 10А

## Функциональность
- 3D позиционирование шара
- PID контроль парения
- Калибровка датчиков
- Консольное управление по UART
- Потоковая передача данных

## Сборка

make -j16 all
